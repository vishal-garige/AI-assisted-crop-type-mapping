# AI-Assisted Crop Type Mapping using Sentinel-2 NDVI and Random Forest

This project performs AI-assisted crop type mapping using Sentinel-2 satellite imagery, NDVI (Normalized Difference Vegetation Index), and a Random Forest machine learning classifier. It integrates Remote Sensing, GIS, and Python to analyze vegetation health and classify crop regions.

## Features

* Sentinel-2 satellite data processing
* NDVI generation using Red and Near Infrared (NIR) bands
* Raster preprocessing and cleaning
* AI-based crop classification using Random Forest
* Visualization of vegetation distribution and classified maps
* Geospatial analysis using Python and GIS

## Tech Stack

* Python
* Rasterio
* GeoPandas
* NumPy
* Matplotlib
* Scikit-learn
* QGIS

## Applications

* Precision Agriculture
* Crop Monitoring
* Land Cover Classification
* Vegetation Health Analysis
* Remote Sensing & GIS
